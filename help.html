Справка по библиотеке HexAStar

Инициализация

f_define_hexastar_functions()
Инициализирует DLL и определяет все функции. Вызовите эту функцию один раз в начале игры до использования других функций библиотеки.

Функции работы с картами

f_map_create(width, height)
Создает гексагональную карту с заданными размерами.
Возвращает: дескриптор карты (0 при ошибке)
width - количество гексов по оси q
height - количество гексов по оси r
Все ячейки инициализируются стоимостью 1.0 (проходимые).
f_map_destroy(дескриптор_карты)
Удаляет карту и освобождает память.
Возвращает: 1.0 (успех), 0.0 (ошибка)
дескриптор_карты - дескриптор, полученный от f_map_create
f_map_destroy_all()
Удаляет все созданные карты из памяти.
Возвращает: 1.0 (успех)
Полезно при смене уровня игры.
f_map_width(дескриптор_карты)
Возвращает ширину карты в гексах (ось q).
дескриптор_карты - дескриптор, полученный от f_map_create
f_map_height(дескриптор_карты)
Возвращает высоту карты в гексах (ось r).
дескриптор_карты - дескриптор, полученный от f_map_create
f_map_get_info(дескриптор_карты)
Возвращает строку с информацией о карте.
дескриптор_карты - дескриптор, полученный от f_map_create
Формат: "ID:X, Size:WxH, Cells:C"

Функции работы с ячейками

f_map_set_cell(дескриптор_карты, q, r, значение)
Устанавливает стоимость прохождения ячейки.
Возвращает: 1.0 (успех), 0.0 (ошибка)
дескриптор_карты - дескриптор, полученный от f_map_create
q, r - осевые координаты (0..ширина-1, 0..высота-1)
значение - стоимость прохождения:
  • < 0.0 = непроходимая стена
  • = 0.0 = бесплатный проход
  • = 1.0 = стандартная стоимость по умолчанию
f_map_get_cell(дескриптор_карты, q, r)
Возвращает стоимость ячейки карты.
Возвращает: стоимость_ячейки или -1.0 (ошибка)
дескриптор_карты - дескриптор, полученный от f_map_create
q, r - осевые координаты
f_map_fill(дескриптор_карты, значение)
Заполняет всю карту указанным значением стоимости.
Возвращает: 1.0 (успех), 0.0 (ошибка)
дескриптор_карты - дескриптор, полученный от f_map_create
значение - стоимость для всех ячеек
f_map_fill_region(дескриптор_карты, начало_q, начало_r, конец_q, конец_r, значение)
Заполняет прямоугольную область карты указанным значением.
Возвращает: 1.0 (успех), 0.0 (ошибка)
дескриптор_карты - дескриптор, полученный от f_map_create
начало_q, начало_r - начальные координаты области (включительно)
конец_q, конец_r - конечные координаты области (включительно)
значение - стоимость для ячеек области

Функции работы с путями

f_path_create()
Создает объект для хранения пути.
Возвращает: дескриптор пути (0 при ошибке)
Путь изначально пустой.
f_path_destroy(дескриптор_пути)
Удаляет объект пути и освобождает память.
Возвращает: 1.0 (успех), 0.0 (ошибка)
дескриптор_пути - дескриптор, полученный от f_path_create
f_path_destroy_all()
Удаляет все созданные пути из памяти.
Возвращает: 1.0 (успех)
f_path_find(дескриптор_карты, дескриптор_пути, старт_q, старт_r, цель_q, цель_r, разрешить_стены, игнорировать_стоимость)
Выполняет поиск пути по алгоритму A* на гексагональной сетке.
Возвращает: > 0.0 (длина пути), 0.0 (путь не найден), -1.0 (ошибка)
дескриптор_карты - дескриптор карты для поиска
дескриптор_пути - дескриптор пути для сохранения результата
старт_q, старт_r - координаты начального гекса
цель_q, цель_r - координаты целевого гекса
разрешить_стены - разрешить проход через стены:
  • ≥ 1 = true (разрешить, стоимость стены = 1.0)
  • < 0 = false (запретить)
игнорировать_стоимость - игнорировать стоимость ячеек:
  • ≥ 1 = true (все ячейки = 1.0)
  • < 0 = false (учитывать установленную стоимость)
f_path_length(дескриптор_пути)
Возвращает количество точек в сохраненном пути.
Возвращает: длина_пути или -1.0 (ошибка)
дескриптор_пути - дескриптор, полученный от f_path_create
f_path_point_q(дескриптор_пути, индекс)
Возвращает координату q точки пути.
Возвращает: координата_q или -1.0 (ошибка)
дескриптор_пути - дескриптор, полученный от f_path_create
индекс - индекс точки пути (0..длина-1)
f_path_point_r(дескриптор_пути, индекс)
Возвращает координату r точки пути.
Возвращает: координата_r или -1.0 (ошибка)
дескриптор_пути - дескриптор, полученный от f_path_create
индекс - индекс точки пути (0..длина-1)

Пример использования

// Инициализация f_define_hexastar_functions(); // Создание карты 50x50 гексов map = f_map_create(50, 50); // Установка стен for(i = 0; i < 50; i += 1) { f_map_set_cell(map, i, 25, -1.0); // горизонтальная стена } // Создание объекта пути path = f_path_create(); // Поиск пути от (0,0) до (49,49) с обходом стен length = f_path_find(map, path, 0, 0, 49, 49, 0, 0); // Чтение найденного пути if(length > 0) { for(i = 0; i < length; i += 1) { q = f_path_point_q(path, i); r = f_path_point_r(path, i); // Использование координат q,r... } } // Очистка f_path_destroy(path); f_map_destroy(map);
Система координат: используются осевые координаты (q,r) для гексов с ориентацией "pointy-top".
Для отображения на экране требуется преобразование в пиксельные координаты.
