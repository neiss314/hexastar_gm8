# Hex A* pathfinding DLL

## Описание

Библиотека реализует поиск пути между двумя точками на **гексагональной карте (hex grid)** с использованием алгоритма **A\***.

Предназначена для использования в играх на движке **GameMaker 8.1 / 8.2**.

---

## Возможности

### Поиск кратчайшего пути

Библиотека строит маршрут между начальной и конечной клеткой карты с учётом:

- стоимости перемещения по клеткам  
- наличия непроходимых клеток  
- геометрии гексагональной сетки  

Результатом работы является **последовательность клеток**, образующих путь.

---

### Работа со стоимостью клеток

Каждая клетка карты может иметь **собственную стоимость перемещения**.

Это позволяет моделировать различные типы поверхности:

- обычная земля  
- труднопроходимая местность  
- дороги  
- болота  
- вода и т.п.

Алгоритм выбирает путь с **минимальной суммарной стоимостью**, а не просто самый короткий по количеству шагов.
### Поддержка непроходимых клеток

Клетки могут быть помечены как **непроходимые** (стены, препятствия).

Алгоритм автоматически строит путь **в обход таких областей**.

При необходимости можно включить режим, в котором **препятствия игнорируются**.

---

### Возможность игнорировать стоимость клеток

В специальном режиме алгоритм может **игнорировать стоимость клеток** и искать путь только по **геометрическому расстоянию**.

Это полезно для:

- быстрого поиска приблизительного маршрута
- анализа доступности точек
- отладки карт
- летающих врагов

---

### Сглаживание маршрутов

При построении пути учитывается **направление движения**.  
Это уменьшает вероятность появления длинных прямых участков, что делает маршруты **более естественными**.

---

## Особенности реализации

- оптимизация для работы с **большими гексагональными картами**
- не требуется сложных структур данных со стороны пользователя
- работа с обычной **двумерной картой клеток**
- допуск **гибкой настройки поведения поиска пути**
